# 標準ノートテンプレート

## 出力ファイル命名規則

```
notes/obsidian/LectureX_Lab Y_トピック名.md
```

- `notes/obsidian/Lecture3_Lab3_CNN_Image_Classification.md`
- `notes/obsidian/Lecture5_Lab5_RNN_Sequence_Models.md`

## 各セクションの生成ルール

### 本文セクション

- スライドの章立てに沿ってセクションを作る
- 各スライドの「定義・説明・図・例」を日本語でまとめる
- 図・表がある場合 → Markdownの表か箇条書きで再現する

### 表を使うべき場面

| 場面 | 例 |
|------|----|
| 複数の概念・手法を比較するとき | 多クラス vs 2値分類 |
| パラメータ一覧をまとめるとき | `nn.Conv2d` の引数 |
| 入出力・フォーマットを示すとき | 画像のサイズ変化 |
| 用語・定義をまとめるとき | 重要用語まとめ表 |

### LectureとLabの橋渡し

スライドで説明された概念が、Labのどのコードに対応するかを明示する：

```markdown
> **Lecture → Lab**: スライドの「畳み込み演算」= Colabの `nn.Conv2d(...)` に対応。
```

コードブロックには必ず概念コメントを付ける：

```python
# [概念: 畳み込み層] フィルタサイズ3×3、パディング1でサイズを保持
self.conv1 = nn.Conv2d(in_channels=1, out_channels=32, kernel_size=3, padding=1)
```

### 専門用語の扱い

| 種別 | 書き方 | 例 |
|------|--------|----|
| 専門用語 | 日本語 `英語` | 過学習 `overfitting` |
| 関数・クラス名 | バッククォートのみ | `nn.Conv2d` |
| ライブラリ名 | バッククォートのみ | `PyTorch` |
| モデル名 | そのまま英語 | VGG16, ResNet |
| 一般的な英語略語 | 日本語展開 + 略語 | CNN（畳み込みニューラルネットワーク） |

### 曖昧・不明な内容

```markdown
> ※ スライドに記載なし。要確認。
```

```markdown
> **補足（Lab）**: ここからはColabノートブックの補足内容。
```

## 出力品質チェックリスト

### 構造
- [ ] ファイル名が命名規則に従っている（`LectureX_LabY_トピック名.md`）
- [ ] 目次がすべてのセクションにリンクしている（`[[#...]]` 形式）
- [ ] 必須セクションが6つすべて含まれている

### 内容
- [ ] スライドの章立てに沿っている
- [ ] 定義・重要概念が引用ブロック（`>`）で強調されている
- [ ] LectureとLabの橋渡しコメントがある
- [ ] コードブロックに「概念」コメントが付いている

### 言語
- [ ] 専門用語に英語が併記されている
- [ ] 推測による補完をしていない（不明点は明示されている）
- [ ] 重要用語まとめ表がある

### 復習セクション
- [ ] 復習ポイントのチェックボックスがある
- [ ] 混同しやすいポイントの比較表がある
- [ ] 理解チェック問題が3問以上ある（解答例つき）

### Obsidian
- [ ] タグが末尾にある（`#科目 #Lecture-X #Lab-Y`）
- [ ] `[[#セクション名]]` 形式の内部リンクが機能する
